# Triage labels

工程类 skills 使用五个标准分诊角色。这个文件把标准角色映射到本仓库 GitHub Issues 中实际使用的 label 字符串。

| Skill 标准角色 | 本仓库 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 描述已经足够清楚，可以交给 AFK agent |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当 skill 提到某个分诊角色时，使用上表右侧对应的 GitHub label。

如果以后仓库改用不同 label 名称，只修改“本仓库 label”这一列。
