# ChatLuna Plugins

ChatLuna 插件集合，围绕 Koishi 插件能力组织可复用的 ChatLuna 扩展。

## Language

**分群排除模板 key 列表**:
按群限定的模板排除清单，用于表达某个群额外不可使用的 meme 模板 key。
_Avoid_: 分群屏蔽列表, 分群黑名单, 群模板排除

**用户趋势指标变化**:
用户当前综合好感、长期好感或对话次数与上一次记录不同。短期好感、关系、昵称和最后互动时间的变化不单独构成用户趋势指标变化。
_Avoid_: 状态变化、分量变化

**历史线延伸**:
仪表盘展示用户好感度历史时，将最近一次记录的趋势指标水平延伸到当前趋势锚点日期。这个延伸点只用于展示，不代表新增了一条用户快照。
_Avoid_: 补写快照、生成历史

**用户好感展开区**:
仪表盘中展开单个用户时展示的个人好感趋势区域，核心曲线为综合好感、长期好感和对话次数。
_Avoid_: 单独查看的用户好感
