# AAAAACAT-chatluna-plugins

这是一个以 ChatLuna 生态为核心的 Koishi 插件聚合仓库，主要存放我维护的 ChatLuna 相关扩展插件，以及它们共用的基础包。

## 当前插件

- `plugins/chatluna-affinity`
- `plugins/chatluna-meme-generator`
- `plugins/chatluna-thoughts-viewer`
- `plugins/chatluna-toolbox`

## 共享包

- `packages/shared-chatluna-xmltools`: ChatLuna XML 工具调用相关的共享解析与运行时能力
- `packages/shared-nav`: 多个插件控制台页面共用的导航组件
- `packages/tsconfig`: 仓库内 TypeScript 基础配置

- `koishi-plugin-chatluna-schedule`: https://github.com/Sor85/koishi-plugin-chatluna-schedule
